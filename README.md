# Pokemon Project

## Overview

This project is a Java-based Pokémon simulation that allows users to create trainers, manage Pokémon teams, explore regions, and interact with wild Pokémon. The project focuses on object-oriented design, file handling, and game logic development.

## Features

- Create and manage Trainers
- Create and manage Pokémon
- Pokémon evolution system using external files
- Region exploration and trainer movement
- Wild Pokémon generation
- Team management
- File reading for Pokémon and evolution data

## How to Run

1. Open the project in IntelliJ IDEA.
2. Ensure the required files are included:
   - pokedex.txt
   - Evolve.txt
3. Run the main Java file.
4. Follow the menu options displayed in the console.
